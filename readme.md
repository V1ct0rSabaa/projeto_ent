# Meu Projeto Flask

Este é um projeto simples utilizando Flask para criar uma página web que exibe texto Lorem Ipsum, um vídeo do YouTube e um link para a Wikipedia.

## Tecnologias Usadas

- **Flask**: Um microframework para Python, usado para criar aplicações web.
- **Bootstrap**: Utilizado para estilização da página.

## Estrutura do Projeto
1) clone o repositório
git clone https://github.com/seu_usuario/meu_projeto_flask.git
cd meu_projeto_flask
2) Instale as dependências:
pip install -r requirements.txt
3) Execute a aplicação:
py app.py
4) URL do site: 
http://127.0.0.1:5000
